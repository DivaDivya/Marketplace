# 🛒 Marketplace

A full-stack e-commerce web application inspired by platforms like Amazon and Facebook Marketplace — where users can register as customers or sellers, and an admin oversees the entire platform.

---

## 🚀 Features

### 👤 Customer
- Register and log in securely
- Browse and search product listings
- Purchase products from sellers

### 🏪 Seller
- Register and log in as a seller
- List, edit, and manage product inventory
- Track product listings

### 🔐 Admin
- View and manage all registered members (customers & sellers)
- Access system analytics including sales data and platform activity
- Moderate and manage user reviews

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Node.js, Express.js |
| Templating | EJS (Embedded JavaScript) |
| Routing | RESTful API design |
| Frontend | HTML, CSS, JavaScript |

## ⚙️ Getting Started

### Prerequisites
- Node.js (v14+)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/DivaDivya/Marketplace.git

# Navigate into the project directory
cd Marketplace

# Install dependencies
npm install

# Start the server
node app.js
```

Then open your browser and go to `http://localhost:3000`
